# welcome_project_fe

A new Flutter project.

## Getting Started
- under lib -> util, you can add your colors, your icons, to make it reusable for your screens
- under lib -> view -> base, this is where you can create reusable widgets like buttons, etc
- under lib -> api_service.dart, this is where you can write your api calls, to help connect 
- the backend and the frontend together.

## The things that is in the FrontEnd:
- UI responsiveness for PC & Phone
- Calling Backend API from the frontend
- Responsive Navigation Menu (Side Menu/ Sidebar)
- UI components: Card or Table (Either one) Snackbar, Icon and Button.
- Adding assets and images.
